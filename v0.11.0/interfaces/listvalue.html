Interface ListValue

Represents a homogeneous list of values.

Example:

{
    valueType: 'list',
    list: [
        { valueType: 'party', party: 'Alice' },
        { valueType: 'party', party: 'Bob' }
    ]
}

To express values in a more concise way, you can have a look at the ValueHelpers.
